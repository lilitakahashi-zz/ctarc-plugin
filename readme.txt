== Description ==

Publique banners em seu site rápido e fácil, sem precisar entender nada de código. Adicione um banner dentro do seu post.

= Plugin shortcode =

<code>[cta_banner id="XX"]</code>

*Responsivo

== Installation ==

1. Fazer o upload da pasta 'ctarc-plugin' para o diretório '/wp-content/plugins/'.
2. Ativar "ctarc-plugin" na lista de plugins através do menu 'Plugins' do painel WordPress.

== Frequently Asked Questions ==

O que o CTA Banner faz?
